# the dependency array must be added to the test invocation.

# From Table 4.3-A of JVM:
#
#   | FieldType term | Type                                        |
#   |----------------|---------------------------------------------|
#   | B              | byte                                        |
#   | C              | char                                        |
#   | D              | double                                      |
#   | F              | float                                       |
#   | I              | int                                         |
#   | J              | long                                        |
#   | L ClassName ;  | reference -> an instance of class ClassName |
#   | S              | short                                       |
#   | Z              | boolean                                     |
#   | [              | reference -> one array dimension            |
